# Digitful

Digitful is an Astro marketing site for a growth strategy business focused on strategy, acquisition, and automation.

## Scripts

- `npm install` installs dependencies
- `npm run dev` starts the local development server
- `npm run build` creates the production build
- `npm run preview` previews the production build locally

## Design Direction

The project uses a dark, product-style visual system defined in [DESIGN.md](./DESIGN.md).
